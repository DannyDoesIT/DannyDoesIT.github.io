01

Basic VLAN Configuration

Create and assign VLANs on a single switch. Verify intra-VLAN connectivity and confirm traffic isolation between broadcast domains.

VLANs SWITCHING
Objectives
  • Create VLANs 10, 20, and 30 on a 2960 switch
  • Assign access ports to each VLAN
  • Verify hosts in same VLAN can communicate
  • Confirm hosts in different VLANs cannot communicate

02

Trunk Port Configuration

Configure 802.1Q trunking between two switches so that VLAN traffic can pass across the uplink. Verify with show commands.

VLANs TRUNKING SWITCHING
Objectives
  • Configure trunk port between Switch1 and Switch2
  • Allow specific VLANs across the trunk
  • Use show interfaces trunk to verify
  • Test end-to-end VLAN connectivity across switches

03

Inter-VLAN Routing — Router on a Stick

Enable communication between VLANs using a single router interface with subinterfaces. Classic enterprise pattern before Layer 3 switches.

ROUTING VLANs SUBINTERFACES
Objectives
  • Configure router subinterfaces for each VLAN
  • Set encapsulation dot1Q on each subinterface
  • Set trunk mode on the switch uplink port
  • Verify cross-VLAN ping between hosts

04

Static Routing Between Networks

Connect two separate networks via static routes on a pair of routers. Understand next-hop addressing and route table verification.

ROUTING STATIC ROUTES
Objectives
  • Assign IP addresses across a two-router topology
  • Configure static routes with correct next-hop addresses
  • Verify routes with show ip route
  • Test full end-to-end reachability
